# Todo List Testing 

یک پروژه ساده todo list داریم با reactjs . 
<br/> 
<br/> 
دو کامپونت داریم یکی برای form که دو input دورنش قرار گرفته 
<br/>
<br/>
یگ کامپونت به اسم UserList که user های که درون input ها وارد می کنیم به ما نشون میده
